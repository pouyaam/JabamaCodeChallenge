# Project Overview
This Android application demonstrates a GitHub authentication and repository search feature. It leverages clean architecture, MVVM, Jetpack Compose, Coroutines, and unit/UI testing to maintain a robust and scalable codebase.

## Key Features:

GitHub authentication using client ID and secret
Repository search functionality with dynamic typing to search.
Clean architecture principles for better maintainability
MVVM architecture for separation of concerns
Jetpack Compose for declarative UI development
Coroutines for asynchronous operations
and, some unit and UI testing for ensuring the app quality.
#### Prerequisites
- Android Studio Koala Feature Drop | 2024.1.2
- JDK 17+
- Android SDK with latest updates
#### Setup
- Clone the Repository:
```bash
git clone https://github.com/rezazarchi/JabamaCodeChallenge.git
```
- Checkout to the branch `zarchi`
- Let to download dependencies and sync the project become complete
- Have an Android emulator or real device with Android version 7.0 or newer
- Click the "Run" button in Android Studio to build and install the app on a connected device or emulator.
